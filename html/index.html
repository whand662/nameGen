NameGen Documentation

IMPOTANT: must use c++11 to compile!

This is a name generation tool for C++ applications. Often times, a project will need to populate large lists with fictional data for testing and demonstration purposes. Rather than building a name generator specific to each project I built this multipurpose tool to save time. By changing the contents of the text files in ./res you can suit the generator to your own needs. (each line is read as an element)

Just add the whole nameGen directory and #include "./nameGen/NameGen.hpp"
